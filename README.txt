PG Buddy

PG Buddy is a user-friendly platform designed to simplify the process of finding and managing PG accommodations. Our mission is to ensure that everyone has access to comfortable, safe, and affordable living spaces.
Features
•	Quality Listings: All listed properties meet our quality standards.
•	Trusted Community: Built on transparency between PG owners and seekers.
•	Security Focus: A secure platform where users can connect with confidence.
•	Personalized Support: Ensuring a great experience for both owners and seekers.
•	Verified Listings: Ensuring authenticity and quality.
•	Secure Platform: Prioritizing user safety and data protection.

Why Choose PG Buddy?
•	Easy and efficient way to find or list PG accommodations.
•	Offers verified listings to ensure the quality of stay.
•	Community-driven, fostering trust and connection.
•	Secure platform to protect user data.

Tech Stack
•	Frontend: React, Tailwind CSS
•	Backend: Node.js, Express
•	Database: MongoDB
•	Libraries: lucide-react, react-router-dom

Requirements
•	Node.js (v14 or higher)
•	npm (v6 or higher)

Installation Commands
    1.	Clone the repository:
    git clone <repo-url>
    2.	Navigate to the project directory:
    cd pg-buddy
    3.	Install dependencies:
    npm install
    4.	Run the development server:
    npm start
    5.	Build for production:
    npm run build
    6.	Run tests:
    npm test

Project Structure
•	src/components: Reusable UI components.
•	src/pages: Main pages like Home, About, and Contact.
•	src/styles: Custom styles for the application.
•	src/assets: Images and other static files.
